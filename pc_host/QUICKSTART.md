# 快速入门指南

## 第一次运行

### 方法1：使用启动脚本（推荐）

双击运行 `run.bat` 文件，脚本会自动：
1. 检查虚拟环境
2. 安装依赖（如需要）
3. 启动程序

### 方法2：手动运行

```bash
# 1. 进入目录
cd pc_host

# 2. 激活虚拟环境
.venv\Scripts\activate

# 3. 运行程序
python main.py
```

## 基本操作流程

### 1. 连接设备

1. 用USB线连接S800板到PC
2. 点击"刷新"按钮
3. 选择对应的COM口（如COM3）
4. 点击"连接"按钮
5. 看到"已连接"绿色状态

### 2. 设置时间

**方法1：手动设置**
- 在"基础控制"标签页
- 调整时分秒数值
- 点击"设置时间"

**方法2：网络对时（推荐）**
- 切换到"高级功能"标签页
- 点击"NTP网络对时"
- 自动获取并设置精确时间

### 3. 设置日期

- 在"基础控制"标签页
- 调整年月日数值
- 点击"设置日期"

### 4. 设置闹钟

1. 在"基础控制"标签页
2. 调整闹钟时间
3. 点击"设置闹钟"（自动启用）
4. 需要关闭时点击"关闭闹钟"

### 5. 使用数字孪生面板

右侧面板完整镜像S800板状态：

- **查看显示**：实时显示数码管内容
- **查看LED**：显示8个LED的状态
- **虚拟按键**：点击K1-K8按钮，等同于按下板上按键
- **USER1按钮**：点击后自动触发NTP对时
- **USER2按钮**：显示天气信息（需先获取天气）

### 6. 查看通信日志

底部日志区域显示所有通信：
- **蓝色**：发送的命令
- **绿色**：接收的应答
- **橙色**：事件（按键、显示变化等）
- **红色**：错误信息

## 常用功能

### 发送滚动消息

1. 在消息输入框输入文本（≤32字节）
2. 点击"发送消息"
3. S800板数码管会滚动显示

### 控制蜂鸣器

1. 设置时长（10-5000ms）
2. 点击"蜂鸣"
3. S800板蜂鸣器响铃

### 控制LED

1. 输入16进制值（00-FF）
2. 点击"设置LED"
3. 示例：
   - FF = 全亮
   - 00 = 全灭
   - A5 = 10100101（间隔亮）

### 切换显示方向

- 点击"方向LEFT"：正常显示（如12.30.45）
- 点击"方向RIGHT"：反转显示（如54.03.21）

### 昼夜模式

- **DAY模式**：显示全部8位数码管+全部LED
- **NIGHT模式**：仅显示时分4位+仅保留心跳LED

## 扩展功能

### E1 - NTP网络对时

1. 确保PC联网
2. 点击"NTP网络对时"
3. 自动获取标准时间并下发到S800板
4. 或者：S800板按USER1按键，PC自动触发对时

### E2 - 天气获取

1. 确保PC联网
2. 点击"获取天气"
3. 显示当前天气信息
4. S800板按USER2按键可短显天气5秒

### E3 - 昼夜模式

1. 点击"昼间模式"或"夜间模式"
2. S800板和数字孪生面板同步切换
3. 夜间模式节能显示

## 演示功能

### 参数组合演示

在"高级功能"标签页：
1. 选择下拉框中的预设命令
2. 点击"发送选中命令"
3. 演示23种参数组合

### 缩写演示

点击以下按钮演示协议缩写：
- "MIN演示"：`*SET:TIME MIN 30`
- "SEC演示"：`*SET:TIME SEC 45`
- "DISP演示"：`*SET:DISP ON`

### 大小写混合演示

点击以下按钮演示大小写不敏感：
- "混合大小写"：`*SeT:TiMe HoUr 12`
- "全小写"：`*set:time hour 12`

## 查询功能

在"查询命令"标签页：
- 查询日期：`*GET:DATE`
- 查询时间：`*GET:TIME`
- 查询闹钟：`*GET:ALARM`
- 查询显示：`*GET:DISPLAY`
- 查询方向：`*GET:FORMAT`

## 自定义命令

在"查询命令"标签页底部：
1. 输入自定义命令（如`*PING`）
2. 点击"发送自定义命令"或按Enter键
3. 在日志中查看响应

## 日志管理

### 清空日志
点击"清空日志"按钮清除所有日志记录

### 导出日志
1. 点击"导出日志"按钮
2. 选择保存位置
3. 日志保存为纯文本文件

## 状态栏说明

顶部状态栏实时显示：
- **方向**：LEFT/RIGHT（显示方向）
- **模式**：DAY/NIGHT（昼夜模式）
- **闹钟**：已启用/已关闭/响铃中
- **延迟**：通信延迟（ms）
- **运行**：S800板运行时间（秒）

## 常见问题

### Q: 连接不上串口？
A: 
1. 检查USB线连接
2. 确认S800板已上电
3. 检查是否有其他程序占用串口
4. 尝试点击"刷新"重新扫描

### Q: 显示不正常？
A:
1. 检查"状态栏"中的FORMAT设置
2. 确认DISPLAY为ON状态
3. 检查是否在NIGHT模式（仅显示4位）

### Q: 虚拟按键没反应？
A:
1. 确认已连接串口
2. 查看日志是否有发送记录
3. 检查S800板是否返回ERROR

### Q: NTP对时失败？
A:
1. 检查PC网络连接
2. 确认防火墙未阻止
3. 稍后重试
