# 智能联网时钟系统 - PC上位机

## 项目概述

这是智能联网时钟系统的PC上位机程序，实现了与S800板的串口通信、数字孪生镜像显示、扩展功能等完整功能。

## 功能特性

### 核心功能（必做）

1. **串口管理** (P1)
   - 自动扫描并列出可用串口
   - 支持连接/断开操作
   - 后台线程接收数据
   - 状态栏实时显示连接状态、FORMAT、MODE、ALARM、延迟、运行时间

2. **控制面板** (P2)
   - 覆盖所有串口命令的可视化界面
   - 日期时间设置（支持多种参数组合）
   - 闹钟设置与控制
   - 显示控制（开/关、LEFT/RIGHT）
   - 滚动消息发送
   - 蜂鸣器控制
   - LED直接设置
   - 系统命令（PING、RST）
   - 23种参数组合演示
   - 缩写演示（MIN、SEC、DISP）
   - 大小写混合演示

3. **数字孪生面板** (P3)
   - 8位七段数码管镜像显示
   - 8位LED镜像显示
   - 8位按键（K1-K8）可点击
   - USER1/USER2独立按钮
   - 完整双向同步
   - 支持FORMAT RIGHT模式（显示反转）
   - 支持NIGHT模式（仅显示时分4位）
   - 小数点精确显示

4. **收发日志** (P4)
   - 时间戳标记
   - 方向标记（发送/接收/事件/错误）
   - 颜色编码
   - 支持导出

5. **异常处理** (P5)
   - 串口占用检测
   - 解析异常提示
   - 网络超时处理
   - ERROR应答弹窗提示
   - 连接丢失自动检测

### 扩展功能（选做）

1. **E1 - 网络对时**
   - 一键从NTP服务器获取标准时间
   - 自动监听USER1按键并触发对时
   - 显示对时结果和时间

2. **E2 - 天气获取**
   - 自动获取上海天气（温度+天气状况）
   - 通过协议下发到S800板显示
   - 状态显示

3. **E3 - 自动昼夜模式**
   - 支持DAY/NIGHT模式切换
   - NIGHT模式：仅显示时分4位、LED仅保留心跳
   - 数字孪生面板同步切换

## 安装说明

### 环境要求

- Python 3.7+
- Windows 10/11（推荐）

1. 激活虚拟环境（如果已创建）：

## 使用指南

### 1. 连接S800板

1. 将S800板通过USB连接到PC
2. 点击"刷新"按钮扫描串口
3. 在下拉框中选择对应的COM口
4. 点击"连接"按钮

### 2. 基础控制

在"基础控制"标签页中：
- **设置日期时间**：调整年月日时分秒，点击"设置日期"/"设置时间"
- **设置闹钟**：调整闹钟时间，点击"设置闹钟"，点击"关闭闹钟"可关闭
- **显示控制**：控制数码管开关和显示方向（LEFT/RIGHT）
- **发送消息**：输入滚动消息（≤32字节），点击"发送消息"
- **蜂鸣器**：设置时长（10-5000ms），点击"蜂鸣"
- **LED控制**：输入HEX值（00-FF），点击"设置LED"

### 3. 高级功能

在"高级功能"标签页中：
- **NTP对时**：点击"NTP网络对时"从互联网获取精确时间
- **获取天气**：点击"获取天气"获取当前天气信息
- **昼夜模式**：切换DAY/NIGHT模式
- **参数组合**：选择预设命令组合并发送
- **缩写演示**：演示协议缩写功能（MIN/SEC/DISP）
- **大小写演示**：演示协议大小写不敏感特性

### 4. 查询命令

在"查询命令"标签页中：
- 查询当前日期、时间、闹钟、显示状态、方向等
- 发送自定义命令

### 5. 数字孪生面板

右侧面板完整镜像S800板状态：
- **数码管**：实时显示8位数码管内容（包含小数点）
- **LED**：显示8位LED状态（绿色=亮，灰色=灭）
- **按键**：点击K1-K8按键，等同于按下S800板物理按键
- **USER1/USER2**：点击USER1请求对时，点击USER2显示天气

### 6. 通信日志

底部日志区域显示所有通信记录：
- **蓝色**：发送的命令
- **绿色**：接收的应答
- **橙色**：事件报文
- **红色**：错误信息
- **灰色**：系统信息

点击"导出日志"可保存日志到文件。

## 协议说明

### 命令格式

所有命令以`*`开头，支持：
- **大小写不敏感**：`*SET` = `*set` = `*SeT`
- **缩写支持**：`MINUTE` = `MIN` = `MINU` = `MINUT`
- **空格容错**：参数间可有多个空格

### 主要命令

#### 设置命令
```
*SET:DATE YEAR MONTH DATE 2026 06 01
*SET:TIME HOUR MINUTE SECOND 12 30 45
*SET:ALARM HOUR MINUTE SECOND 07 00 00
*SET:ALARM OFF
*SET:DISPLAY ON/OFF
*SET:FORMAT LEFT/RIGHT
*SET:MSG <text>
*SET:BEEP <ms>
*SET:LED <hex2>
*SET:KEY <NAME>
*SET:MODE DAY/NIGHT
```

#### 查询命令
```
*GET:DATE
*GET:TIME
*GET:ALARM
*GET:DISPLAY
*GET:FORMAT
```

#### 系统命令
```
*PING
*RST
```

### 事件报文

S800板主动上报：
```
*EVT:KEY <NAME>          # 按键事件
*EVT:DISP <8字符> <dpHex> # 显示变化（1Hz心跳）
*EVT:LED <hex2>          # LED变化（1Hz心跳）
*EVT:ALARM               # 闹钟响铃
*EVT:ALARM_OFF           # 闹钟停止
*EVT:EDIT <TYPE> <VALUE> # 本地编辑
*EVT:MODE <STATE>        # 模式切换
```

## 技术要点

### 1. 七段数码管显示

采用字符到段码的映射表，支持：
- 数字 0-9
- 字母 A-Z
- 特殊字符（空格、减号、下划线等）
- 小数点独立控制

### 2. 数字孪生同步机制

- S800板每秒发送`*EVT:DISP`和`*EVT:LED`（1Hz心跳）
- PC端根据心跳自动刷新显示，无需重传机制
- 支持FORMAT RIGHT模式下的显示反转
- 支持NIGHT模式下的部分显示

### 3. 虚拟按键实现

- 点击PC面板按键 → 发送`*SET:KEY <NAME>` → S800板等同物理按键
- S800板不再上报`*EVT:KEY`，避免环回

### 4. 串口通信

- 波特率：115200
- 数据位：8
- 停止位：1
- 校验：无
- 行结束符：`\r\n`（兼容`\r`或`\n`）

## 故障排除

### 1. 串口连接失败

- 检查USB线是否连接
- 确认S800板已上电
- 检查是否有其他程序占用串口
- 尝试更换USB端口

### 2. 收不到数据

- 检查波特率是否为115200
- 检查S800板程序是否正常运行
- 点击PING测试连接

### 3. 显示不正常

- 检查FORMAT设置（LEFT/RIGHT）
- 检查DISPLAY状态（ON/OFF）
- 检查是否在NIGHT模式

### 4. 按键无响应

- 确认已连接串口
- 检查日志是否有发送记录
- 检查S800板是否返回ERROR
